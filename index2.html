innerHTML: parse and return html content ,including tags. It can modify the element's structure .
innertext: Returns only visible text, ignoring hidden element (display:none). it does not parse HtML.

Question-4: What is event bubbling in the DOM?

Event bubbling directs an event to its target. It works like this: When an element (like a button) is clicked, an event is directed to the element. If an event handler is set for the element, the event handler is triggered. Then the event "bubbles up" to the elements parent. If an event handler is set for the parent, this event handler is triggered. The event continues to bubble, until it reaches the top (the document element).
